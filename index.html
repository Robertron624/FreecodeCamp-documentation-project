What is Python

Python is a highly popular interpreted, high level and general-purpose programming language. It was designed with an emphasis on code readability(code that is easy to understand for humans) and help programmers to write clear, easy to write and logical code for small and large scale projects.

Python is a dynamically-typed language wich means that unlike other languages like C or Java it doesn't require the explicit declaration of the type of a variable before using it, because it's type is checked during run-time. One of Python biggest feature is that there is a large number of libraries ready to install that extend it's functions, because of that Python as a ton of applications, here is a list of his most common uses:

  • Web development: Python offers multiple tools for back-end web development such as the frameworks: Django, Flask and Pyramid, this tools simplify tasks like content management and database interactions.
  • Game development: Python is loaded with many useful libraries that are useful for game development like Pygame, Pyglet and PyOpengl.
  • Scientific investigation: Thanks to his huge number of libraries Python is one of the must used programming language to analyse and process scientific data with libraries such as: Pandas, NumPy and SciPy.
  • AI and machine learning:Python is the de facto programming language to create and train machine learning and AI models with libraries like: Keras, TensorFlow, PyTorch and Pandas.
  • Web-scrapping: Python is a wonderful tool for extracting large amounts of data from web pages like product prices and job listings wich can be used for market analysis and Search Engine Optimization.

Declaring variables

Variables are containers for storing data values. You do not need to declare variables before using them, or declare their type. Every variable in Python is an object. To declare a variable in Python you need to follow this syntax: variable-name + the symbol "=" + variable value. Here is an example:

> name = "John"
> age = 19

If you then print the variable you will see its value.

> print(name)
> John
> print(age)
> 19
Variable scopes

The part of a program where a variable is accessible is called its scope. There are four major types of variable scope and is the basis for the LEGB rule, meaning: Local, Enclosing, Global and Built-in.

Local scope: Whenever you define a variable within a function, its scope lies ONLY within the function. It is accessible from the point at which it is defined until the end of the function and exists for as long as the function is executing. Which means its value cannot be changed or even accessed from outside the function. Here's a simple example:

> def print_number():
local_number = 5
print(local_number)
> print_number()
> 5

The variable "local_number" only exists inside the variable so if we try to print it without invoking the function it will give us an error.

Enclosing Scope: is a special scope that only exists for nested functions. If the local scope is an inner or nested function, then the enclosing scope is the scope of the outer or enclosing function. This scope contains the names that you define in the enclosing function. The names in the enclosing scope are visible from the code of the inner and enclosing functions. Take this as example:

> def outer():
first_num = 1
def inner():
second_num = 2
print(second_num)
> outer()
> 2

The variable "second_num" cannot be accesed from the "outher()" function because the variable doesn't exist there. However you can acces the variable "first_num" from "inner()" because that variable as already been declared in the upper function. This is an enclosing scope. Outer's variables have a larger scope and can be accessed from the enclosed function inner().

Global scope: This is the easiest scope to understand. Whenever a variable is defined outside any function, it becomes a global variable, and its scope is anywhere within the program. Which means it can be used by any function. For example:

> greeting = "Hello"
> def say_hi():
print(greeting)
> say_hi()
> Hello

Built-in scope: This is the widest scope that exists. This scope contains names such as keywords, functions, exceptions, and other attributes that are built into Python. Data types

Python has the following data types built-in by default:

  • Text: string. Example: name = "Robert"
  • Numeric: integer, float, complex Example: myInt = 3, myFloat = 3.0, myComplex = 3 +2j
  • Sequence: list, tuple. Example: my_list = [1, 2, 3], mytuple = (2, 3, 4).
  • Mapping: dictionary. Example: my_dict = {"name":"albert", "age":25, "job":"doctor"}
  • Set: set, frozenset. Example: my_set = {"banana", "apple", "cherry"}
  • Boolean: bool. Example: isGood = True
  • Binary: bytes, bytearray, memoryview. Example: x = bytes(3)
Function declaration

A function is a block of code wich only runs when it's called, you can pass data, which we call "parameters" into a function. A function can return data as result.

In python a function must be defined with the keyword "def", followed by the name of the function, then two parenthesis and finally a colon. Finally indentation is IMPORTANT in Python if we want a block of code to be part of a function we must indent it to the function declaration. Here's an example:

> def my_function():
print("Hello world")

To call a function and execute it we use the function's name followed by parenthesis:

> my_function()
> Hello world
If...elif...else statements

The if control structure "If..." allows a program to execute instructions when a condition is fulfilled. It follows this syntax: if... condtion...execute or not exceute. For example:

> a = 5
> b = 10
> if b > a:
print("b is greater than a")
> b is greater than a

The "elif" keyword is pythons way of saying "if the previous conditions were not true, then try this condition". Following the example above lets take a look at this one:

> a = 20
> b = 10
> if b > a:
print("b is greater than a")
> elif a > b:
print("a is greater than b")
> a is greater than b

Note that the first condition is not true so it doesn't get excuted, then it continues with the second condition which is true and it executes it.

The "else" keyword catches anything which isn't caught by the preceding conditions. Let's take a look:

> a = 50
> b = 700
> if a > b:
print("a is greater than b")
> elif a == b:
print("a is equal to b")
> else:
print("a is less than b")
> a is less than b

In this case neither the first nor the second condition are true, then if both aren't true then the code inside the else statement is excuted.
